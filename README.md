# Smart Shopping Advisor MCP

**"도움은 AI가, 선택·판단은 사용자가"**

구매할 물품을 입력하면 제품의 특징, 장단점, 구매 시 주의사항과 함께  
각 쇼핑몰별 장단점 및 구매 링크를 제공하는 지능형 쇼핑 의사결정 지원 MCP 서버

<br><br>

## 🎯 프로젝트 소개

**Smart Shopping Advisor MCP**는 온라인 쇼핑 의사결정을 돕는 MCP(Model Context Protocol) 서버입니다.  
제품 특성의 장단점과 쇼핑 사이트별 특징을 종합 분석하여 사용자의 고민을 덜어줍니다.

<br><br>

## ✨ 핵심 기능

### 1. 제품 상세 분석
```
📦 제품 정보 제공
├─ 주요 특징 및 사양
├─ 장점 / 단점 정리
└─ 구매 전 주의사항 (정품 확인, AS 정책 등)
```

<br>

### 2. 쇼핑몰 비교 및 링크 제공
 
```
🛒 지원 쇼핑몰 : 쿠팡, 네이버 쇼핑, SSG, G마켓, 무신사, 알리익스프레스, ...
```

각 쇼핑몰별로 다음 정보를 제공합니다:
- ✅ 장점 (배송 속도, 반품 정책, 가격 경쟁력 등)
- ⚠️ 단점 (품절 가능성, 배송 기간, 가격대 등)
- 🔗 제품 검색 직링크

<br>

### 3. 상황별 쇼핑몰 추천
```
💡 선택 팁 제공
├─ "빠른 배송 필요" → 쿠팡 추천
├─ "가격 비교 중요" → 네이버 쇼핑 추천
├─ "AS 안정성 중시" → SSG / 공식몰 추천
└─ "해외 직구 관심" → 알리익스프레스 추천
```

<br><br>

## 💬 사용 예시

### 예시 1: 기본 제품 조회

**사용자 입력:**
```
마샬 블루투스 스피커 추천해줘
```

<br>

**출력:**
```markdown
# 📦 마샬 블루투스 스피커 구매 가이드

## 제품 특징
- 빈티지 감성의 독특한 디자인
- 마샬 특유의 저음 강조 사운드
- 휴대용(Emberton)부터 대형(Woburn)까지 다양한 라인업

## 장점
✅ 디자인 만족도가 매우 높음
✅ 브랜드 사운드 아이덴티티가 확실함
✅ 내구성이 우수함

## 단점
⚠️ 가격 대비 음질 성능 논란 있음
⚠️ 병행수입 제품의 경우 AS 불편
⚠️ 무게감이 있어 휴대성 제한적

## 구매 전 체크리스트
- [ ] 정품/공식 유통 여부 확인
- [ ] 방수 등급(IPX7 등) 확인
- [ ] 사용 환경(실내/야외) 고려
- [ ] AS 정책 및 기간 확인


## 🛒 쇼핑몰별 비교

| 쇼핑몰 | 검색 링크 | 장점 | 단점 | 추천 상황 |
|-------|-----------|------|------|-----------|
| 쿠팡 |  URL  | 로켓배송, 무료반품 | 인기 모델 품절 잦음 | 빠른 배송 필요 시 |
| 네이버쇼핑 |  URL  | 가격 비교 용이 | 판매자별 정책 상이 | 가격 비교 원할 때 |
| SSG |  URL  | 정품 신뢰도 높음 | 가격대 다소 높음 | AS 중요할 때 |
| 공식몰 |  URL  | 100% 정품, AS 확실 | 할인 거의 없음 | 품질 최우선 |


## 💡 쇼핑몰 선택 가이드

**빠른 배송이 중요하다면** → 쿠팡 (로켓배송 당일/새벽 배송)
**가격을 꼼꼼히 비교하고 싶다면** → 네이버 쇼핑 (여러 판매처 한눈에)
**AS와 정품이 가장 중요하다면** → 공식몰 또는 SSG.COM
**해외 직구로 저렴하게** → 알리익스프레스 (배송 기간 2~4주 소요)
```

## ⚖️ 법적 고지

### 데이터 수집 정책

본 프로젝트는 **웹 스크래핑, 크롤링, 자동화된 데이터 수집을 수행하지 않습니다.**

#### 우리가 하는 것 ✅
- 공개된 쇼핑몰 검색 URL 생성
- 일반적인 제품 특성 정보 제공
- 쇼핑몰별 공개된 정책 정보 요약

#### 우리가 하지 않는 것 ❌
- 실시간 가격 정보 수집
- 재고 현황 크롤링
- 리뷰 데이터 스크래핑
- 개인정보 수집 및 저장

<br>

```
모든 쇼핑몰 링크는 제품명 기반의 공개 검색 URL 템플릿을 조합하여 생성되며, 자동화된 접근이나 데이터 추출 목적이 아닙니다.
```
